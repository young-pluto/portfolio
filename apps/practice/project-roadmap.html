WEEK 1: Async Foundation CRITICAL FOR BOTH PROJECTS

1. Promises (11.2) - Firebase & Payment APIs

From your code:

  • new Promise(), resolve(), reject()
  • .then(), .catch(), .finally()
  • Promise chaining
  • Promise.all(), Promise.race()

2. Async/Await (11.8) - Modern Payment Processing

From your payment system:

  • async functions
  • await keyword
  • Converting promises to async/await

3. Error Handling (10.1) - Payment Failures

From your code:
try { await auth.signInWithEmailAndPassword(email, password); } catch (error) { alert(`Error: ${error.message}`); }
  • try, catch, finally
  • Error handling with promises
  • Custom error messages

4. Modules (13.1) - Your App Structure

Your IIFE module pattern:

  • IIFE (Immediately Invoked Function Expressions)
  • Module privacy
  • Public/private methods
