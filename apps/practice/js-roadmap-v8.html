1. CORE JAVASCRIPT FUNDAMENTALS

Variables & Data Types

  • let, const, var (scope differences)
  • Primitive types: string, number, boolean, null, undefined, symbol
  • Reference types: object, array, function
  • Type checking: typeof, instanceof

Functions

  • Function declarations vs expressions
  • Arrow functions: () => {}
  • Parameters: default, rest (...args), destructuring
  • Return statements
  • this binding in different contexts
  • IIFE (Immediately Invoked Function Expressions)

Objects

  • Object creation: literal, new Object(), Object.create()
  • Property access: dot notation, bracket notation
Essential Methods:
  • Object.keys()
  • Object.values()
  • Object.entries()
  • Object.assign()
  • Spread operator {...obj}
  • Destructuring: const {name, age} = user
  • Computed property names

Arrays CRITICAL

Essential Methods You Must Know:
  • push(), pop(), shift(), unshift()
  • slice(), splice()
  • map(), filter(), reduce(), forEach()
  • find(), findIndex(), some(), every()
  • includes(), indexOf(), join(), sort()
  • concat(), spread operator [...arr]
  • Array destructuring: const [first, second] = arr

2. CONTROL FLOW & LOOPS

  • if/else, switch, ternary operator
  • for, for...in, for...of, while, do...while
  • break, continue

3. ERROR HANDLING

  • try, catch, finally
  • throw statements
  • Error objects and custom errors
  • Promise error handling

4. ASYNCHRONOUS JAVASCRIPT CRITICAL

Promises

  • new Promise(), resolve(), reject()
  • .then(), .catch(), .finally()
  • Promise chaining
Promise API Methods:
  • Promise.all()
  • Promise.race()
  • Promise.allSettled()

Async/Await

  • async functions
  • await keyword
  • Error handling with try/catch
  • Converting promises to async/await

Timers

  • setTimeout(), clearTimeout()
  • setInterval(), clearInterval()

5. DOM MANIPULATION CRITICAL FOR YOUR PROJECTS

Element Selection

  • document.getElementById()
  • document.querySelector(), document.querySelectorAll()
  • document.getElementsByClassName(), document.getElementsByTagName()

Element Manipulation

  • element.textContent, element.innerHTML
  • element.setAttribute(), element.getAttribute()
ClassList Methods:
  • element.classList.add()
  • element.classList.remove()
  • element.classList.toggle()
  • element.classList.contains()
  • element.style.property
  • element.dataset

Element Creation & Modification

  • document.createElement()
  • element.appendChild(), element.removeChild()
  • element.insertBefore(), element.replaceChild()
  • element.remove()
  • document.createDocumentFragment()

Event Handling

  • addEventListener(), removeEventListener()
Event Object Properties:
  • event.target
  • event.preventDefault()
  • event.stopPropagation()
  • Common Events: click, input, change, submit, DOMContentLoaded
  • Event delegation
